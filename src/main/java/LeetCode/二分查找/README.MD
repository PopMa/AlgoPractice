## 二分查找的适用:
比0(n)还要快的算法复杂度，那必须就是0(logn)了（这里说的是在一般的面试情况下）
那么O(logn)就必然要考虑二分的方法来做了。
一般都会与一些排序的序列、在一段有规则的序列等情况中找到符合某个条件的位置/值。
